Overview

This document details a unique Tic Tac Toe web application that provides a diverse range of gaming experiences through multiple modes, online multiplayer functionality, and adaptable gameplay settings.

Challenge AI with adjustable difficulty levels from beginner to expert.

Customizable Gameplay

Adjust grid sizes, themes, and player names for personalized experience.

Core Features

Multiple Game Modes

  • Player vs AI (PvAI): Engage with an intelligent AI opponent offering adjustable difficulty levels.
  • Player vs Player Offline (PvP Offline): Classic local multiplayer on a single device.
  • Player vs Player Online (PvP Online): Compete against players globally in real-time matches.

User Account System

  • Secure registration and login with email/password credentials
  • Personalized user profiles with unique usernames
  • Comprehensive game statistics tracking (wins, losses, draws)
  • Global leaderboards to compare performance

AI Difficulty Levels

  • Easy: Makes random moves with basic strategy elements
  • Medium: Strategic thinking with some unpredictability
  • Hard: Uses minimax algorithm for optimal play

Customizable Game Options

  • Adaptable grid sizes (3x3, 4x4, 5x5)
  • Light and dark visual themes
  • Customizable player names
